# Smart Fraud Detection System

## 📌 Overview
The Smart Fraud Detection System is a machine learning-based application that detects fraudulent financial transactions. It helps identify suspicious activities using transaction data and provides real-time predictions.

---

## 🚀 Features
- Detects fraud and genuine transactions
- Uses Machine Learning (Random Forest)
- Handles imbalanced data
- Simple Streamlit user interface
- Real-time prediction with probability score

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Matplotlib
